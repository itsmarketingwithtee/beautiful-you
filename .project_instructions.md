# Beautiful You — Mental Health App

## Brand
- **Name:** Beautiful You
- **Logo:** Pink-purple intertwined hand-hearts
- **Colors:** Primary Pink `#E88DAE`, Primary Purple `#7B5EA7`, gradient `['#E88DAE', '#7B5EA7']`, bg `#FAF5FB`

## Architecture
- **Backend:** NestJS + Prisma + PostgreSQL, deployed at `beautiful-you.abacusai.app`
- **Frontend:** React Native Expo SDK 54, expo-router file-based routing
- **Auth:** Email/password + guest mode

## Key Design Decisions
- Community pills: uniform height rounded rectangles, just background color changes on active (pink/purple rotating)
- Community has a pinned welcome post always at top
- Journal is Notes/diary style (title + body + optional mood), NOT chat-style
- "AI Support" renamed to **"Heart to Heart"** — warmer, less robotic
- Crisis responses: empathetic tone, never say "glad you reached out" — validate feelings first
- Resource Helper chatbot: client-side resource search, crisis keyword detection directs to 988/741741/Talk to Us
- 31 daily wellness tips rotating by day of year
- Hero images on Home, Resources, Community headers

## DB Models
- User, UserSettings, Affirmation (57 seeded), UserFavorite, MoodEntry, PushToken, ChatMessage, CommunityPost/Reaction/Report, JournalEntry (title, body, mood)
